# Proyecto-Web-Ejemplo

## 🚀 Cómo clonar y ejecutar el proyecto localmente

Para tener una copia de este proyecto funcionando en tu computadora, sigue estos sencillos pasos:

1. **Clonar el repositorio:**
   Abre tu terminal (PowerShell o Git Bash) en la carpeta donde quieras guardar el proyecto y ejecuta:
   ```bash
   git clone [https://github.com/Miguezx21/Proyecto-Web-Ejemplo.git](https://github.com/Miguezx21/Proyecto-Web-Ejemplo.git)

   Entrar a la carpeta del proyecto:

Bash
cd Proyecto-Web-Ejemplo
Ejecutar el código:

Al ser una página web estática básica, no necesitas instalar dependencias pesadas.

Simplemente ve a tu explorador de archivos, entra a la carpeta y haz doble clic sobre el archivo index.html. Se abrirá automáticamente en tu navegador web (Opera GX, Chrome, Edge, etc.).

Si usas Visual Studio Code, te recomendamos instalar la extensión Live Server, hacer clic derecho sobre el index.html y seleccionar “Open with Live Server” para ver los cambios en tiempo real mientras programas.

🌿 Estructura de Ramas del Proyecto
En este repositorio gestionamos el flujo de trabajo mediante las siguientes ramas:

main (Rama Principal): Es la rama de producción. Contiene el código limpio, estable y verificado. Todo lo que esté aquí es la versión final que vería un usuario.

Funcionalidad actual: Estructura HTML base (index.html) conectada a una hoja de estilos moderna (styles.css) que centra y maqueta el diseño de manera elegante.

master (Rama de Historial/Legado): Utilizada inicialmente durante la configuración local del entorno. Actualmente todo el desarrollo principal se ha unificado en la rama main para seguir los estándares actuales de GitHub.
